# 常见样式声明

1. color
元素内部的文字颜色
预设值：
三原色：
光学三原色: rgb(0,0,0);
hex表示: #000000;

2. background-color

3. font-siz元素内部文字的尺寸大小
> px：像素
> em：相对单位，相对于父元素的字体大小
每个元素必须有字体大小，如果没有声明，则直接使用父元素的字体大小，如果父元素也无声明，则使用基准字号（浏览器默认字号）
> user agent，用户代理（浏览器)

4. font-weight
> \<strong>元素：表示强调

5. font-family
sans-serif:非衬线字体

6. font-style
> \<i>元素：默认样式是倾斜字体，实际使用时，通常用它表示一个图标

7. text-decoration
> \<a>
> \<del>元素：表示错误的内容
> \<s>元素：表示过期的元素

8. text-indent
首行文本缩进

9. line-height
每行文本的高度
单行文本时，设置行高为容器的高度，可以让单行文本垂直居中
行高可以设置为纯数字，表示相对于当前元素的字体大小

10. width

11. height

12. letter-spacing
文字间隙

13. text-align
文字水平排列方式
